Root0emir Announces Cyrethium

August 23, 2025 Root0Emir

Project founder Root0Emir announced Securonis's new name as Cyrethium. The new name was chosen as a unique, unregistered name derived from the combination of "Cyber" and "Ethereum".

With the new name, the project officially moved to the cyrethium.org domain. Additionally, Root0Emir announced that regular Warrant Canary publications will be made following Cyrethium's release.

Key highlights of the announcement:

  • Official transition to cyrethium.org domain
  • Regular Warrant Canary publications planned
  • Focus on threat analysis and system attack detection
  • About 75% of project tools designed from scratch
  • Development primarily in Python and Bash for easy Linux management

The developed software is analyzed by Claude 4.0 Opus to detect and fix potential security vulnerabilities and performance issues. For existing Securonis users, Cyrethium Lite will be released as a simpler, minimal version with high usability.

Securonis Linux Project Name Change

August 10, 2025 Development

The Securonis Linux project recently underwent a name change, and the GitHub repository was archived during this process.

The developer was forced to make the name change due to conflicts with a similar brand. The similarity caused both high copyright risk and SEO problems. Users frequently reported in forums that they couldn't reach the correct sources and had difficulty accessing information due to SEO issues.

Reasons for the name change:

  • Brand conflict with existing similar names
  • High copyright and trademark risks
  • SEO problems affecting user access
  • User confusion in finding official resources

For these reasons, the project decided to change its name to Cyrethium, ensuring a unique and conflict-free brand identity moving forward.

Root0Emir Linux Magazine Interview

July 4, 2025 Linux Magazine

Linux Magazine featured Debian-based Securonis Linux and an interview with the distribution's developer Root0Emir (Emir). Running on the MATE desktop environment, Securonis stands out with its wide range of security and privacy tools.

In the interview, Emir discussed his entry into the open-source world and his interest in security-focused Linux distributions. After realizing the lack of tools that met his needs, he decided not only to use existing tools but also to develop his own open-source cybersecurity tools.

Interview highlights:

  • Securonis menu includes 11 original tools alongside MATE's native tools
  • Command-line based management tools like Password Manager and System Knight
  • User-friendly interface combined with advanced security functions
  • Vision to make Securonis functional for both daily users and privacy-focused professionals

Emir's vision is built on making Securonis a functional platform for both daily users and privacy-focused professionals. This approach makes Securonis stand out not just as a security distribution, but as an innovative project in the open-source world.

Linux Security: Securonis Raises Privacy Bar

June 24, 2025 Linux Security

LinuxSecurity.com thoroughly reviewed Securonis Linux and presented the distribution as an important option for privacy and anonymity-focused system administrators. According to the article, Securonis routes all internet traffic through the Tor network by default, and this approach sets the distribution apart from its competitors.

Securonis offers technical flexibility as a Debian Testing-based distribution. However, this also requires users to accept some performance and technical challenges. The system elevates security and privacy to the highest level with built-in tools like DNSCrypt, I2P Router, and FireScorpion browser.

Key features highlighted:

  • All traffic routed through Tor network by default
  • Built-in DNSCrypt, I2P Router, and FireScorpion browser
  • Ideal environment for system administrators, pentesters, journalists, and activists
  • Live OS testing recommended before installation

While not designed for daily use, Securonis stands out as a noteworthy option for professionals who prioritize privacy and security. LinuxSecurity.com recommends trying it first on live OS to test the tools before installation.

Mark Rabideau's Securonis Review

June 19, 2025 Mark Rabideau

Mark Rabideau, one of the recognized names in the Linux world, thoroughly reviewed Securonis Linux and evaluated the distribution with praise. According to Rabideau, Securonis is not just a security-focused operating system; it's also a fast, lightweight, and impressive platform designed for daily use.

Securonis was developed based on Debian Trixie, thus built on a stable foundation. However, what makes the distribution stand out is its privacy and security-focused design. Measures like AppArmor profiles, kernel hardening, secure default settings, and sandboxing are integrated throughout the system.

Key review points:

  • Doesn't create difficulties for users in daily use
  • Offers both performant and practical experience through minimal design
  • Enriches user experience with custom-developed tools and scripts
  • Thoughtful approach in design: desktop themes, service configurations, and system settings prepared with great care

Mark Rabideau states that Securonis is a young but impressive project and recommends everyone who values digital privacy to try it. The distribution's balanced structure in terms of both performance and security makes it noticeable in the Linux ecosystem.

Securonis Linux Listed on Distrowatch

May 23, 2025 Distrowatch

Securonis Linux had submitted its application to be listed on Distrowatch. After a 3-week wait, it entered the waiting list and was added to the database 4 days later.

Distrowatch is one of the most well-known platforms in the Linux world for users to discover different distributions, track their popularity, and stay informed about current developments. Every year, millions of users discover new systems through Distrowatch.

Significance of this milestone:

  • Global visibility beyond Turkey
  • Access to millions of Linux users worldwide
  • Important milestone for international recognition
  • Path to acceptance as a reliable distribution

Securonis's presence on Distrowatch ensured the project's visibility not only in Turkey but worldwide. In short, the Distrowatch success is just a beginning for Securonis Linux. This step became an important milestone on the path to global recognition and acceptance as a reliable distribution.
