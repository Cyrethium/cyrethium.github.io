About Cyrethium

Cyrethium is a Debian-based Linux distribution focused on privacy, security, and threat analysis.
It can route all system traffic through the Tor network with CyrethOnion; blocks non-Tor exit traffic and implements comprehensive configurations against potential data leaks.
It also isolates applications and websites by using a separate Tor circuit for each of them, preventing cross-traffic correlation and improving anonymity.

With tools, it can manipulate network traffic to produce fake traffic and alter packet timing, making exit node monitoring and analysis attacks much more difficult.

Cyrethium doesn't just offer passive protection; it provides system defense capabilities with active threat detection tools.
It includes powerful tools such as Tor node analysis, node attack detection, suspicious traffic inspection, non-Tor connection detection, and system anomaly detection.
Users can detect and prevent potential attacks in advance with node analysis tools.

For multi-layered anonymity, Cyrethium offers I2P network and DNSCrypt-Proxy support.
Secure web browsing is provided with Tor and I2P supported hardened custom browsers:

Hardened Firefox: Provides high security and stability in daily use.

Amnesic Firefox: Focuses on strict privacy and security, automatically deleting all traces when closed.

With Cyrethium Jails, every file and application can be easily sandboxed.
Hardened kernel and system settings provide high resistance against exploits, rootkits, and trojans.
Thanks to advanced security tools, malware can be detected and removed from the system both in daily use and in-depth technical analyses.

The distribution offers a comprehensive privacy ecosystem with more than 40 custom-designed tools.
Some usage traces are automatically cleaned at each system shutdown, ensuring continuous user privacy protection.

Features

Tor Integration

Allows forcing all system traffic to be routed through the Tor network with the CyrethOnion tool. Non-Tor traffic is rejected and potential leaks are minimized

I2P Integration

You can browse the internet anonymously with built-in I2P support and a browser with pre-configured I2P settings.

Hardened System

The system includes many additional security settings and kernel hardening configurations. The system is highly resistant to many cyber attacks and exploits.

Balance in Daily Usage

The hardening settings are well-balanced and their effect is barely noticeable in daily use. It provides a beautiful interface with customized MATE

Custom Designed Tools

The system contains nearly 40 custom-designed privacy, security, crypto, and threat analysis tools. Each one runs completely from source code and is embedded in the system.

Cyrethium Monero Wallet

An easy-to-use, completely locally running Monero CLI-based wallet. It works integrated with CyrethOnion to provide crypto security.

Tor Node Security

The system provides special tools for Tor node analysis detection. Payload attacks, suspicious nodes, node attacks, and traffic leaking outside of Tor can be detected.

Debian-based

The system uses the highly stable and secure Debian base. It doesn't use any non-Debian repositories; all updates depend solely on Debian repositories.

Completely Open Source

Cyrethium is completely open source and open to all kinds of audits. Every Cyrethium Project, including the website, is completely free software

Browser Security

The system uses custom-designed Hardened Firefox and Amnesic Firefox. Browser attacks are blocked at a very high rate and it's completely ad-free. Amnesic Firefox leaves no traces behind.

MATE

The system has a modern and easy-to-use MATE desktop. Cyrethium offers a pleasant interface and ease of use with its customized MATE desktop.

Network and System Security

Cyrethium comes with many network and system security tools. Network analysis and defense can be performed, and malware and attack detection can be done on the system
